# Simple sum of integers
```c
    int n = 100
    int s = 0
    for (i = 0; i < n; i++){
        s += i;
    }
    printf("%s\n", s);
```

```python
    n = 100
    s = 0
    for i in range(0,100):
        s += i
    print(s)
```
